# <p style="color:#FFD1DC;">BARTECHIES – METABOLIC RESET FOR TECHIES</p> 

## <p style="color:cyan;">Overview</p>

BarTechies is an AI-powered metabolic health platform designed specifically for people who spend long hours working at desks, especially developers, students, and office professionals.

Many tech professionals sit for 8–10 hours daily, which can negatively impact metabolic health. BarTechies helps solve this problem by providing personalized metabolic insights and lifestyle optimization using Non-Exercise Activity Thermogenesis (NEAT).

Instead of relying on intense workouts, the platform focuses on micro-movements, standing intervals, and smart activity patterns to improve metabolic activity throughout the day.

**Live Demo:**
https://bartechies.netlify.app/


## <p style="color:cyan;">Problem Statement</p>

Modern work environments are increasingly sedentary, particularly in the tech industry. Long hours of sitting can lead to:

* Reduced metabolic flexibility.
* Increased fatigue and stress.
* Higher risk of insulin resistance.
* Weight gain and reduced physical activity.

Most professionals do not have enough time for gym workouts, making it necessary to optimize metabolic health during working hours.

## <p style="color:cyan;">Our Solution</p>

BarTechies introduces an intelligent system that helps desk workers improve metabolic health through small but effective lifestyle adjustments.

The platform uses NEAT-based analysis to recommend personalized activity patterns that can be integrated into daily routines.

Users can monitor their activity levels and receive AI-driven suggestions to improve metabolic efficiency without interrupting their workflow.

## <p style="color:cyan;">Key Features</p>

### <p style="color:skyblue;">AI Metabolic Coach</p>
Provides intelligent suggestions to improve daily metabolic activity based on user habits and lifestyle.

### <p style="color:skyblue;">NEAT Score Calculation</p>
Evaluates the user's Non-Exercise Activity Thermogenesis to determine metabolic activity levels.

### <p style="color:skyblue;">Desk Movement Optimiztion</p>
Encourages small movements such as standing intervals, stretching, and posture changes.

### <p style="color:skyblue;">Smart Sitting vs Standing Protocol</p>
Recommends optimal sitting and standing intervals to reduce metabolic slowdown.

### <p style="color:skyblue;">Metabolic Health Insights</p>
Tracks user habits and provides actionable insights for long-term metabolic improvement.

## <p style="color:cyan;">NEAT Score Calculation</p>

The platform calculates a NEAT Score based on several lifestyle factors including:

* Daily desk hours
* Standing time
* Activity patterns
* Age

This score helps estimate how active a person is outside of structured exercise.

## <p style="color:cyan;">How it works</p>

1. Assess
    + Users enter basic lifestyle information to calculate their metabolic baseline.
2. Analyze
    +  The system evaluates the user's activity level using NEAT-based analysis.
3. Recommend
    + AI generates personalized activity and lifestyle recommendations.
4. Improve 
    + Users gradually increase their metabolic activity through small daily changes.

## <p style="color:cyan;">Tech Stack</p>

### <p style="color:skyblue;">Frontend</p>
The user interface of BarTechies was built using core web technologies to keep the platform lightweight and fast.

  + **HTML5** – Used to structure the website and create the layout of different sections such as the homepage, calculator, and informational content.
  + **CSS3** – Used for styling the application, designing the UI, managing layouts, and making the website responsive across different devices.
  + **JavaScript (JS)** – Handles the interactive parts of the application including user inputs, dynamic calculations, and real-time updates of the NEAT score.

### <p style="color:skyblue;">Application Logic</p>
The main functionality of the project relies on JavaScript logic that processes user inputs and generates metabolic insights.

   + **NEAT Score Algorithm** – Calculates Non‑Exercise Activity Thermogenesis based on user inputs such as desk hours, standing time, and activity levels.
   + **Dynamic Input Handling** – JavaScript event listeners detect changes in user inputs and update the results instantly.
   + **Client-Side Processing** – All calculations are performed directly in the browser without requiring a backend server.

### <p style="color:skyblue;">UI/UX Design</p>
The interface is designed to be simple and accessible so users can quickly interact with the calculator and understand their results.

   + Responsive Layouts using Flexbox and modern CSS.
   + Minimal and clean design for better usability.
   + User-friendly input forms for quick data entry.
